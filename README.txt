---Overview---

Implementation of random forests from David Sontag NYU machine learning course: http://cs.nyu.edu/~dsontag/courses/ml16/slides/lecture11.pdf.

---Requirements---

* numpy: https://docs.scipy.org/doc/numpy/user/install.html
* matplotlib: https://matplotlib.org/users/installing.html

---Example---

1) Change dir to where random_forests.py is.

2) Run this in a python terminal:

from random_forests import RandomForests
rf = RandomForests()
rf.generate_example()

See the function generate_example() in random_forests.py
